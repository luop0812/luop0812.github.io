## About Misha

Misha is a cluster named after [Misha Mahowals](https://www.mahowaldprize.org/), a pioneer in neurocomputation who invented silicon retina. 

Misha currently inlcudes the following nodes: 

* 14 H100 nodes, each with four 80GiB h100 GPU cards
* 6 A100 nodes, each wtih four 80GiB a100 GPU cards
* 6 A40 nodes, each with four 48GiB a40 GPU cards
* 2 big memory nodes, each with 2TiB memory 
* 26 CPU nodes

Additional nodes will be added in the future. 

## Working Environment
Misha provides the same working environment as the other YCRC clusters, except for a few differences. This site provides documentations on those specific features. 
For more general information, please refer to the [YCRC user guide](https://docs.ycrc.yale.edu/).
